def add_edge(mat, i, j):

def display_matrix(mat):
  
    # Display the adjacency matrix
    for row in mat:
        print(" ".join(map(str, row)))

def add_edge(adj, i, j):
    adj[i].append(j)
    adj[j].append(i)  # Undirected

def display_adj_list(adj):
    for i in range(len(adj)):
        print(f"{i}: ", end="")
        for j in adj[i]:
            print(j, end=" ")
        print()

# Create a graph with 4 vertices and no edges
V = 4
adj = [[] for _ in range(V)]

# Now add edges one by one
add_edge(adj, 0, 1)
add_edge(adj, 0, 2)
add_edge(adj, 1, 2)
add_edge(adj, 2, 3)

print("Adjacency List Representation:")
display_adj_list(adj)

** DFS **
To traverse all vertices in a graph, often needs a auxilury data structure, a stack. Because we need to have the ability to return to the previous state.
Stack has FILO and LIFO, using LIFO in a stack will bring us to the previous state. Alot of times we will not use a stack, but instead use recursion call stack.
To travese nodes, move the node to the stack, remove the node from the stack, check if the node has been visited and mark the visited node. Find adjacent nodes, and put them 
all into the stack, remove the top node from the stack, check if the node has been visited, mark the node. Add the adjacent node to the new node to stack. Rinse and repeat.
